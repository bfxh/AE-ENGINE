## 1. 项目概述

wasteland 是一个 Rust 3D 游戏引擎项目，专注于高性能 3D 渲染和物理模拟。

## 2. 技术栈

| 技术 | 用途 |
|------|------|
| Rust | 主要编程语言 |
| Godot 4 | 游戏引擎基础 |
| GDExtension | Rust-Godot 绑定 |
| Bevy ECS | 实体组件系统 |

## 3. 代码规范

- 使用 `cargo fmt` 格式化代码
- 遵循 Rust 官方编码规范
- 添加适当的文档注释（`///`）
- 编写单元测试和集成测试

## 4. 工作流程

```
开发 → 测试 → PR → 审查 → 合并 → 发布
```

## 5. 特殊规则

- 所有提交必须通过 `cargo test`
- 代码审查至少需要1人批准
- 性能关键路径需要性能基准测试
